# 🎬 影视资源分享站

基于 Astro + Fuwari 构建的影视资源导航站，自动同步更新最新影视剧集的网盘下载链接。

## 📌 博客定位

这是一个专注于影视资源分享的导航站点，提供：

- **最新影视资源**：持续更新热门电影、电视剧的网盘下载链接
- **多网盘支持**：整合夸克网盘、迅雷云盘、百度网盘、UC网盘等多种下载方式
- **实时状态追踪**：标注影视资源状态（热播中/已完结/持续更新）
- **影视信息展示**：提供影片简介、演员、导演、年份等详细信息

## 📚 内容来源

通过 Google Sheets 管理影视资源数据，自动生成并发布文章，确保内容的及时性和准确性。

## 🚀 技术架构

- **前端框架**：Astro
- **主题模板**：Fuwari
- **部署平台**：Cloudflare Pages
- **数据管理**：Google Sheets

每篇影视资源文章包含：

- 📷 影视封面
- 📖 剧情简介
- 🎭 演职人员信息
- 💾 多网盘下载链接
- 📊 资源状态（更新中/已完结）
